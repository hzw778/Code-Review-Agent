# Code Review Agent 开发手册

> 本手册是项目开发的路线图，每完成一个任务就把 `[ ]` 改成 `[x]`。
> 严格按阶段顺序推进，不跳阶段。每完成一个阶段才能进入下一阶段。

---

## 项目概述

- **项目名**：Code Review Agent（智能代码审查 Agent）
- **定位**：面向开发团队的 AI 代码审查助手，接入 Git 仓库，对 PR/Commit 自动做静态审查，输出可执行修改建议；并支持对代码库自然语言问答。
- **技术栈**：Java 17 + Spring Boot 4.0.6 + Spring AI 2.0.0-RC1 + MySQL + Redis + Elasticsearch + JGit + JavaParser
- **大模型**：GLM-4.5-air（OpenAI 兼容接口）
- **数据访问**：JPA Repository（非 MyBatis Mapper）
- **日志规范**：SLF4J + Logback，中文详细日志，详见 agent.md 规则8

---

## 项目分层结构说明

```
service/              # 服务层接口
  impl/               # 服务层实现类（如 RepoServiceImpl、ReviewServiceImpl）
repository/           # JPA 数据访问层
  entity/             # JPA 实体类
```

注意：本项目使用 Spring Data JPA，数据访问层是 Repository 接口（继承 JpaRepository），不需要 MyBatis 的 Mapper。

  - [ ] AstAnalysisTool（包装阶段3）

- [ ] 在 Agent Loop 每一步加入工具调用日志和 LLM 调用日志
- [ ] 接口：
